# Component Documentation

## Layout Components
- **Header**: Main navigation and search.
- **Footer**: Site-wide links and info.
- **Layout**: Wrapper component for consistent page structure.

## Product Components
- **ProductForm**: Handles variant selection and Add to Cart.
- **QuickAddButton**: AJAX-based button for adding products from collection pages.
- **ProductSpecs**: Detailed technical specifications for products.

## Cart Components
- **CartMain**: The full cart view.
- **CartSummary**: Price breakdown and checkout button.
- **CartLines**: Individual line items in the cart.

## UI Elements
- **Button**: Base button component with variants.
- **Input**: Styled input fields.
- **Modal**: Accessible modal dialogs using Headless UI.
